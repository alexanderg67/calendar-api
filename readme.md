# Project Title

API for aplication react-calendar. 

## Description

Endpoints URL: 

http://hostname:port/api/v1/users \
http://hostname:port/api/v1/events 
## Getting Started

 

### Installing
* run: npm install
* Create file .env file and type MONGO_URI, JWT_SECRET and PORT( for api)
Example: \
MONGO_URI=mongodb://localhost/EVENTS  \
JWT_SECRET=ggg \
PORT=5000  

### Executing program

* Run: npm start
